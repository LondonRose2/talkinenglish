What is CEFR?

CEFR is the Common European Framework Reference for measuring language levels. There are 3 main levels and each one includes 2 sublevels and guidelines to measure each level.

  • Basic (A1, A2)
  • Intermediate (B1, B2)
  • Advanced (C1, C2)
